## Predict Discrete Emotion

Use this endpoint to analyze a short audio clip and return a predicted emotion with a confidence score.

---

## Request

**POST** `/predict-discrete`

### Headers

| Name        | Type   | Required | Description              |
|-------------|--------|----------|--------------------------|
| `api_key`   | string | ✅ Yes   | Your API key (can be in header or form field) |

### Form Data

| Field      | Type     | Required | Description                        |
|------------|----------|----------|------------------------------------|
| `file`     | file     | ✅ Yes   | Audio file (.wav, .mp3)            |
| `api_key`  | string   | ✅ Yes   | API key (if not sent in header)    |

---

## Response

```json
{
  "emotion": "happy",
  "confidence": 0.91
}
